# sparkleOS ✨

right now this is just a simple "hello world" bootloader that stays in real mode

![boot screenshot](screenshots/boot.png)

## requirements

nasm - to compile

```
brew install nasm
```

qemu - to emulate

```
brew install qemu
```

## compile

```
make
```

## run on qemu

```
make run
```
